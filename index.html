IoT Air Quality & Environmental Monitoring Dashboard

SGP40 Sensor Integration Overview

Project Overview

This project presents an advanced IoT-based environmental monitoring system built around the ESP32 microcontroller. The device continuously measures a wide range of atmospheric and air-quality parameters, providing real-time data for research and practical applications.

My primary responsibility was developing the firmware for the SGP40 sensor, which reads raw gas sensor data and generates a meaningful TVOC (Total Volatile Organic Compounds) index. This index estimates indoor air quality based on VOC concentration.

Beyond TVOC readings, the system collects temperature, humidity, pressure, altitude, CO₂ concentration, particulate matter across multiple sizes, particle counts, and typical particle size. All data are structured in JSON and sent to a cloud API for seamless dashboard integration.

System Overview

The ESP32 collects the following measurements during each cycle:

  • Temperature (°C) – Ambient air temperature.
  • Humidity (%) – Relative moisture content.
  • Pressure (hPa) – Atmospheric pressure levels.
  • Altitude (m) – Calculated using pressure variations.
  • CO₂ (ppm) – Carbon dioxide concentration.
  • Particulate Matter (PM1.0, PM2.5, PM4.0, PM10) – Number of particles per cm³ per size class..
  • Particle Count (NC0.5, NC1.0, NC2.5, NC4.0, NC10) – Mass concentration of airborne particles across size ranges..
  • Typical Particle Size – Average detected particle diameter.
  • Raw VOC Signal (SGP40) – Gas sensor output.
  • TVOC Index – Total volatile organic compound index.
  • Timestamp – Exact measurement time.
  • Device ID – ESP32 identifier.

Combined, these readings create a complete snapshot of environmental changes and air-quality conditions for each recorded moment.

Key Features

  • SGP40 TVOC Processing: Converts raw readings into reliable TVOC index.
  • Comprehensive Dataset: Collects 20+ parameters per measurement cycle.
  • Reliable Connectivity: Auto Wi-Fi reconnection and cloud JSON uploads.
  • Dashboard Ready: Data structured for instant visualization.
  • Extensible System: Supports adding sensors and IoT integration.
