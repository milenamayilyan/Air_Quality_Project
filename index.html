SGP40 Sensor Integration Overview

Project Overview

This project presents an advanced IoT-based environmental monitoring system built around the ESP32 microcontroller. The device continuously measures atmospheric and air-quality parameters and sends structured data to the cloud.

My primary responsibility was developing the firmware for the SGP40 sensor, converting raw VOC readings into a meaningful TVOC index.

System Overview

  • Temperature, Humidity, Pressure, Altitude
  • CO₂ concentration
  • Particulate Matter & Particle Count
  • SGP40 Raw VOC Signal
  • TVOC Index
  • Timestamp & Device ID

Key Features

  • SGP40 raw → TVOC processing
  • JSON-based cloud uploads
  • Dashboard-ready visualization
  • Extensible IoT architecture
